# 사주 계산 규칙

계산 로직을 구현하기 전에 이 규칙을 따를 것. 도메인 규칙이 애매하면 `saju-master` 서브에이전트에게 검증을 요청한다.

## 기본 테이블

천간 (10): 갑(甲,목+) 을(乙,목-) 병(丙,화+) 정(丁,화-) 무(戊,토+) 기(己,토-) 경(庚,금+) 신(辛,금-) 임(壬,수+) 계(癸,수-)

지지 (12): 자(子,수) 축(丑,토) 인(寅,목) 묘(卯,목) 진(辰,토) 사(巳,화) 오(午,화) 미(未,토) 신(申,금) 유(酉,금) 술(戌,토) 해(亥,수)

60갑자: 천간과 지지를 순서대로 짝지어 순환 (갑자, 을축, 병인, ... 계해)

## 반드시 지킬 규칙

1. 년주 경계는 입춘: 양력 1월 1일이나 음력 설이 아니다. 입춘(대략 2월 4일, 해마다 다름) 이전 출생은 전년도 간지.
2. 월주 경계는 절기: 각 월은 12절(입춘·경칩·청명·입하·망종·소서·입추·백로·한로·입동·대설·소한)로 시작.
3. 자시 처리: 23:00~00:59 출생의 날짜 귀속은 야자시/조자시 유파가 갈린다. 구현 시 정책을 상수로 명시하고 주석으로 남길 것.
4. 절기 시각은 천문 데이터: 근사 공식으로 절기를 계산하면 경계 케이스가 틀린다. 만세력 테이블 데이터를 쓸 것.

## 구현 위치

- 계산 코어는 `packages/saju-core`(없으면 생성)에 순수 TypeScript 함수로. UI 의존성 금지.
- 모든 계산 함수는 vitest 테스트와 함께 작성. 경계 케이스(입춘 직전/직후, 자시) 테스트 필수.
